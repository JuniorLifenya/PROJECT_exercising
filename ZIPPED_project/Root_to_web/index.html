Anharmonic Oscillator & TDSE with RK4

This static site bundles all code, data and plots for my Phys291 project. To run the simulations and regenerate the results, follow the instructions in README.md.

Directory layout

  • code/ — all the C++ source files.
  • data/ — the raw .dat/.txt outputs (wavefunctions, spectra, norms…).
  • plots/ — PNGs generated by ROOT and scripts.
  • README.md — build & run instructions.

How & to run


cd submission/code
g++ -std=c++17 -I ../eigen-3.4.0 Main.cpp $(root-config --cflags --libs) -o oscillator
./oscillator
root -l -b -q plot_wave_and_density.C
