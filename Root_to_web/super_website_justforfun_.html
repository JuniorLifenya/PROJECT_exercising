The code is executed by the following line: [root -l -b -q MAIN_CPP_CODES/E_vs_eigenvalues_using_root_plot.C]. In the same folder the code plot_wave_and_density.C generates the two plots bellow. That code is run by a similar command: [root -l -b -q MAIN_CPP_CODES/plot_wave_and_density.C ]
